# SubsonicFS

A filesystem generated from the Subsonic REST API to expose playlists as a local filesystem
(eg to rsync onto an audio player via USB etc..)

## Installation

    $ gem install subsonicfs

## Usage

For usage instructions see

    $ subsonicfs -h
